Zarra Studios / Marcus Zarra Resume

Marcus Zarra, owner of Zarra Studios LLC has been developing software since the late 1980s.  His most recent experience is as follows:

iPhone Development

In addition to its own titles, Zarra Studios has developed or contributed to the following iPhone projects:

  1. •Bullfrog Touch -- Marcus Zarra worked with Jon Trainer to produce Bullfrog for the iPhone.  This game is currently available on the AppStore.

  2. •Mobile Roadie -- Marcus Zarra worked with Fluidesign to develop the initial release of the Mobile Roadie platform.

  3. •Bible -- Marcus Zarra worked with Life Church to develop the current release of Bibe, one of the most popular applications on the iPhone platform.

  4. •My Wine Trail Guide -- Marcus Zarra worked with Momentum Media LLC to develop this fantastic guide to all of the wineries in the United States.

  5. •uSonar -- Marcus Zarra worked with Urban Interactive in concert with Jon Trainer to produce this application for the iPhone.  This application is currently in review to be released on the AppStore.

  6. •FrontPocket -- Zarra Studios wrote this application for Updatable as a iPhone front end to BackPack a 37 Signals Web Application.  This application is current available on the AppStore.

Writing

In addition to the development efforts listed above, Marcus Zarra is involved in the following writing projects:

  1. •Core Data -- Marcus Zarra is writing a book focused on Apple’s Core Data framework for The Pragmatic Programmer.  This book is currently in Beta and expected to be available in print in the Spring of 2009.

  2. •Core Animation -- Marcus Zarra is working with Matt Long to write this book focused on Apple’s Core Animation framework for Addison Wesley.  This book is currently in development and is expected to be in print in the Spring of 2009.

  3. •Cocoa Is My Girlfriend -- Marcus Zarra is a co-author of this wildly successful blog focused on everything Cocoa related.

Previous Relevant Experience

1996-2004 Java Development

Marcus Zarra has developed software for Fortune 500 companies in the following sectors:

  1. •Telecommunications -- Marcus Zarra has developed security software for MCI, Verizon Business and ICG.

  2. •Banking -- Marcus Zarra was part of the team that developed Norwest Bank’s 5 minute loan by phone software.  Norwest Bank has since been purchased by Wells Fargo.

  3. •Health Care -- Marcus Zarra was part of the team that developed the Health / Customer portal for Blue Cross / Blue Shield of Minnesota.

2004-Present Zarra Studios
