iWeb Buddy 1.6 Released!

 
It has been a while since the last release but 1.6 is now available!  The primary focus of this release is to be Lion compatible.  However there are some goodies in  here for everyone.

As always, if you run into any issues with this release, please let us know.



Changes since last build:

  1. •Improved OS X 10.7 Lion support.

  2. •Social bookmarks are now available on all page types.

  3. •Updated the German localization so that the functionality now matches the English version.

  4. •Added an introductory window to help new users understand the application.

  5. •iWeb Buddy is now disabled when iWeb is running to prevent conflicts and other issues.

  6. •Various cosmetic changes.

You can download the latest from here.

Please enjoy this release of iWeb Buddy and report any issues you may find on our new support forum at http://help.zarrastudios.com.
