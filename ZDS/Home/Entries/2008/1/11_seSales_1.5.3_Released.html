seSales 1.5.3 Released

 

seSales 1.5.3 is a maintenance release focused on correcting some minor issues with the 1.5 release. Included in this release are the following items:

Corrections

  1. •Fixed issue with unresolved invoices window not appearing.

  2. •Changed logic for items with custom input. If you add an item it will create a new item. But you can still change the quantity of an item using the appropriate menu item.

  3. •Items with custom input can now have variable quantities

  4. •Fixed issue with search bar in customer management

  5. •Updated the magnetic card reader to handle some edge case card readers

  6. •Corrected a minor issue with the PayPal payment processor

  7. •Corrected a permissions issue with the Discount Management for users with Entry Edition licenses

  8. •Corrected issue with bar codes that contained a dash

  9. •Cleaned up currency localizations in the invoice windows

  10. •Corrected a rare rounding error with some tax rates

  11. •Corrected a scrolling issue on invoices that are opened from the All Invoices window

  12. •Corrected a data issue when rental items are removed from an invoice

  13. •Fixed an issue with the custom input sheet on invoices when the close button is clicked.

  14. •Fixed an issue with key bindings not being saved.

This release is free for all registered used and can be downloaded directly from the links on the right.
