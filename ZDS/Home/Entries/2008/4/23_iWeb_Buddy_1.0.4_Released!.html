iWeb Buddy 1.0.4 Released!

 

Enhancements


Body bottom tags

Like Header Scripts, It is now possible to inject one or more scripts into the bottom of every page on your site.  This feature is avaialble via the Site settings.


German Localization

iWeb Buddy is now localized for German.


Corrections


Copied Domains

A few users have run into an issue linking their multiple domains when they had created them by copying the default domain that comes with iWeb.  This issue has bee resolved and you can now link in your domains.


New Domains

Some users were reporting an issue with creating new Domains inside of iWeb Buddy.  This issue has been corrected.


Wrong Social Bookmarking Icon

iWeb Buddy was injecting the wrong icon for del.cio.us.  This has been addressed.  To get the updated icon republish your site via iWeb and then re-process the site in iWeb Buddy.

This update is free for all registered users and can be downloaded from here.
