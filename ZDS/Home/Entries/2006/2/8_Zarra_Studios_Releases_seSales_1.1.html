Zarra Studios Releases seSales 1.1

 

Zarra Studios has updated its seSales point of sale software for Mac OS X. This latest version includes inventory import and export abilities, changes to the activity reports and several other requested features.

seSales is designed to be an affordable point of sale system for businesses that choose to use OS X and expect their business software to blend seamlessly with their choice of computer hardware.

seSales costs $129.00 for the entry edition and $249.00 for the standard edition. Printed versions are also available. A demo is available from Zarra Studios web site. seSales works with Mac OS X 10.4 or later. Currently seSales is PPC only but a universal binary will be available soon.

Thursday, February 9, 2006
