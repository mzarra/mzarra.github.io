Zarra Studios Releases seSales 1.3.3

 

Zarra Studios has updated its seSales point of sale software for Mac OS X. This update includes the following enhancements:

  1. •Addition of Address Book synchronization.

  2. •Addition of Barcode scanning directly from an invoice.

  3. •Addition of QuickBooks invoice export.

  4. •Added time settings to the Discount window.

  5. •Discounts can now be set to start and stop at the hour and/or minute of a day.

In addition, the following items were corrected:

  1. •Correction of an issue with Purchase Order printing.

  2. •Corrected an issue that allowed unposted purchase orders to receive shipments.

  3. •Corrected an issue with the URL being passed after an issue has been submitted.

  4. •Corrected an issue that allowed orphaned addresses to exist in the database.

  5. •Corrected an issue that prevented categories from being deleted.

  6. •Added warning to un-posted purchase order receipts.

  7. •Corrected resizing issue for purchase order receipts.

  8. •Update to search fields. All search fields should operate properly.

  9. •Correction to Discount window. Archived items are now filtered out.

  10. •Correction to vendor window, productIDs are now displayed correctly.

  11. •Correction to purchase orders, productIDs are now displayed correctly.

  12. •Vendor Management window resize issue corrected.

  13. •Corrected several inconsistencies with data updates. All windows except purchase orders and invoices should update in real time.

  14. •Corrected an issue with database updates when a customer has more than two words in their name.

  15. •Corrected an issue with employee counting that was causing an abnormal application termination.

This update is free for all users.

Tuesday, April 18, 2006
