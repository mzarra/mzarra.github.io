seSales 1.4 Released!

 

We are proud to announce that seSales 1.4 is released!  This is a free upgrade to existing customers.  This release completed the rental items release and is feature complete for the rental feature.

The following is a list of the major changes in this release:

Enhancements

  1. •Inventory auto archive
    It is now possible to flag an inventory item for automatic archival.  When the item runs out of stock, seSales will automatically flag it as archived.

  2. •Customer Notes
    Customer notes have been added to the customer management section of seSales.  It is now possible to add as many notes to a customer record as needed and they are recorded by the date entered.

  3. •Default Invoice Note
    In a company's properties, a default invoice note can be set.  Once set, it will display on every invoice that does not have a custom note defined.

  4. •Admin approval of instant discounts
    In a company's properties it is possible to set a flag that will require admin approval on all instant discounts when employee management has been activated.

  5. •Add Magnetic Card Reader Panel
    If you have a USB magnetic card reader, seSales can now utilize it for reading gift cards and credit cards.

  6. •Rental Rate Multiplier
    Rental rates can now be defined with multiple units such as $5.00 per 3 hours if desired.

  7. •Default Rental Rate
    For all rentals, a default can be set.  This default will be used in the column display of the main window in item management.

  8. •Default Profit
    A default profit percentage can be set in the company's properties.  Once set any new items added will use this setting to calculate either the retail price or the vendor price.  This calculated price can still be overridden.

  9. •Purchase Units
    In the vendor management window, it is now possible to define bulk purchases from a vendor.  For instance, when a box of candy bars are purchased as a single unit from the vendor but sold as multiple units, this can now be properly displayed and handled in the vendor management window.

  10. •Rental Renewal Invoices
    The handling of rental renewals has been altered.  When a rental is renewed it now uses a new invoice instead of re-using the existing invoice.

  11. •Employee Hours
    In the employee management window, it is possible to edit the hours entered by an employee.  

  12. •Customizable Payment Options
    In the properties window, it is now possible to add additional payment options in addition to the existing options.

  13. •Rental Rate Display
    The item management window can now display the rental rate as an optional column.

  14. •Restricted Items
    A new palette has been added to the item management window which allows the definition of item restrictions.  These item restrictions can be a simple confirmation window or can require the cashier to input a value such as a birth date or driver's license number.

  15. •Address Book Syncing
    Address Book syncing is now disabled by default and can be re-enabled in the properties window.

Reports

  1. •Labor Report
    The employee hours report has been revised.  A start and end date can now be selected as well as the employee to report on.

  2. •Payments Report
    A new report has been added to show all payments received over a given time period.  This report will include any authorization codes entered into the system and/or check numbers.

  3. •Report Export
    All reports can now have their primary data exported to a comma separated file for import into another application such as a spreadsheet or database.

  4. •Activity Reports
    The activity reports have been redesigned to reflect gross and net sales as well as taxes collected.  These activity reports now accurately report refunds.

  5. •Instant Discounts Report
    A new report has been added to show all instant discounts given over a specific period of time.  This report will also show the employee and admin who authorized the discount.

  6. •Inventory by Category Report
    The inventory report can now be generated only for specific categories.

Issues

  1. •Returns On Credit Accounts
    When an item return is completed against an invoice that was charged to store credit, the refund will now accurately refund to store credit instead of cash.

  2. •Refund Processing
    Refund processing has had several corrections made to it.  Items will now return to inventory if the flag is set in the company properties.  The drawer will now accurately reflect any pay-outs for refunds.

  3. •Archived Items In Inventory Reports
    Archived items will no longer appear in inventory reports

  4. •New Item Creation
    An issue with new item creation while a search is active has been corrected.  When a new item is created, the search field will be cleared.

  5. •Rental Agreement Duplication.
    This has been corrected.  Rental agreements will only be printed once per invoice.

  6. •Bulk Price Change Freeze Corrected
    The bulk price change window will no longer freeze seSales while working with a large data set.
